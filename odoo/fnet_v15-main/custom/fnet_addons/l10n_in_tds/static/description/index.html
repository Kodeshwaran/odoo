Overview

This module is developed for Companies registered in India to manage Tax Deducted at Source (TDS) feature on Vendor Bills. It helps to post TDS entries to G/L account and generate report for filing TDS

Key Features


Preloaded TDS sections with details about nature of payment and deduction percentages.

Set TDS applicability in the Vendor record.

Deduct TDS against the Vendor Bill depending on TDS applicability.

Post TDS entries to the defined G/L account.

TDS Register tree view report that can be used for filing TDS.

TDS Sections Configuration

Standard TDS rates are pre-loaded with this module which can be modified as required.
Goto the menu, Invoicing > Configuration > TDS > Sections

Define TDS for Vendors

TDS applicability against a Vendor is defined in the Vendor record. Key inputs are PAN number, TDS section and Type to indiciate whether the Vendor is a Company or Individual.

Deduct TDS against Vendor Bill

Once defined in the Vendor record, the system will try to deduct TDS automatically on validation of Vendor Bill. This can also be blocked depending on the context of the Vendor Bill. TDS amount will be shown separately in the total section of Vendor Bill.

On validation, the TDS amount will be posted to the corresponding G/L account. The TDS register can be viewed and exported from the reporting section. This data can be used for filing TDS on the government web portal.
